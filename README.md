# Contact Manager Application

## Overview

The Contact Manager is a full-stack web application that allows users to manage their contacts, including adding, searching, editing, deleting, and marking contacts as blacklisted. This README provides the setup instructions for both the frontend and backend, including the technologies used and key features.

## Features

- **Add Contacts**: Create new contacts with details such as name, phone number, and email.
- **Edit Contact Information**: Update existing contact details.
- **Delete Contacts**: Remove contacts from the list.
- **Retrieve All Contacts**: Get a list of all saved contacts.
- **Search Contacts by Keyword**: Find contacts using keywords like name or phone number.
- **Manage Blacklist**:
  - **Add to Blacklist**: Add specific contacts to the blacklist.
  - **Edit Blacklist Entry**: Update details of existing blacklist entries.
  - **Delete from Blacklist**: Remove contacts from the blacklist.
  - **Retrieve Blacklist**: Get a list of all blacklisted contacts.
